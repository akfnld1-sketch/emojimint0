사용 설명서
1) 캐릭터를 대분류 → 소분류 → 세분화 순서로 선택합니다.
2) 감정·상황 세트, 생성 개수, 배경 테마를 선택합니다.
3) 필요하다면 고급 옵션에서 의상, 색상, 컨셉, 소품을 직접 지정합니다.
4) ‘프롬프트 생성하기’를 누른 뒤, 각 항목 복사 또는 전체 프롬프트 복사를 이용하세요.

캐릭터 선택

대분류 → 소분류 → 세분화 캐릭터를 순서대로 골라주세요.

고급 옵션 (옷 / 색상)

기본값은 자동입니다. 필요할 때만 의상 스타일과 색상을 직접 선택해 세부 연출을 조절하세요.

감정·상황 세트

일상 / 직장 / 연인 / 크리스마스 / 설날 / 연말 세트 중 하나와 생성 개수, 배경 테마를 선택합니다.

프롬프트 결과

각 프롬프트 오른쪽 복사 버튼을 누르면 색이 바뀌며, 한 번에 전체 복사도 가능합니다.
